    快速上手

    基础静态评估只需要 macOS 和 Python 3.11+。VirusTotal、ClamAV 和 Tart 都是可选增强能力，可以稍后再配置。

    1

    初始化并启动

    在项目目录执行以下命令。项目当前没有第三方 Python 运行时依赖，但仍会创建独立虚拟环境。

    ./scripts/setup.sh
./scripts/run.sh

    然后打开 http://127.0.0.1:8765。

    2

    配置 VirusTotal（可选）

    注册或登录 VirusTotal，在账户的 API Key 页面取得密钥，然后到“环境与配置”中保存。CanUInstall 只查询 SHA-256 的已有报告，不上传文件。

    打开 VirusTotal API Key 页面 · 查看 API 文档与使用限制

    3

    启用 Tart 动态分析（可选）

    需要 Apple Silicon Mac、Homebrew 和较多磁盘空间。下面的命令会安装 Tart、下载 macOS 基础镜像，并在专用镜像中安装 osquery。

    ./scripts/setup.sh --with-dynamic

    镜像下载可能达到数十 GB。完成后，“环境与配置”应显示 canuinstall-runtime、osquery 和 eslogger 可用。

    查看 Tart 官方快速入门

    4

    启用本地杀毒扫描（可选）

    安装 ClamAV 后会自动增加本地已知恶意内容扫描。未安装不会阻止其他评估。

    ./scripts/setup.sh --with-clamav
    关于 requirements.txt

    当前后端只使用 Python 标准库，因此文件里没有需要下载的包。保留该文件是为了让安装命令、编辑器和自动化流程都有明确入口；未来增加依赖时也能保持兼容。
